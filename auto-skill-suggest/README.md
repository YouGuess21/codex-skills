# Auto Skill Suggest

Find reusable skills hidden in prior Codex sessions.

Use this when you have repeated similar work across multiple sessions and want to turn those repeated patterns into shareable, portable skills.

Typical input:

- one or more session IDs
- or a time window such as last 7 days or last 30 days

Typical output:

- a ranked list of candidate skills
- or, if requested, newly scaffolded skill folders

This skill is intentionally light on scripts and prefers self-contained skill generation.
