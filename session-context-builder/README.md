# Session Context Builder

Distill prior Codex sessions into a reusable `context.md`.

Use this when you want a fresh session to inherit the important decisions, constraints, paths, and open questions without rereading full session history.

Typical input:

- one or more session IDs
- or a time window such as last 30 days
- (optional) topic to focus on in those sessions

Typical output:

- a concise `context.md`

This skill keeps the result durable and strips out conversational noise.
